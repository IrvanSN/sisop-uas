KETENTUAN PROGRAM:

FORMAT PENULISAN : 
([OPERAND] [OPERATOR] [OPERAND] [OPERATOR] [OPERAND])
ex : (1 + 2 + 3)

FORMAT OPERATOR :
Gunakan x untuk perkalian
Gunakan : untuk pembagian
Gunakan + untuk penjumlahan
Gunakan - untuk pengurangan
Gunakan % untuk modulus (sisa pembagian)
Gunakan ^ untuk eksponensial
